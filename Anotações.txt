React pode ser considerado biblioteca ou framework. Mas (de acordo com a explicação do professor) é mais correto considerar um framework

Framework serve:
Para criar um modelo de trabalho para facilitar na criação de interfaces. Usa-se para reaproveitar código
Fornece recurso de suporte para quase todos os navegadores
Facilita o Data Biding (*)
Facilita o gerenciamento de eventos

React, Vue e Angular são os frameworks mais usados

O React cria uma cópia do HTML para apenas ele conseguir alterar. Evita que plugins ou extensões alterem os arquivos sem q ele tenha o controle. Através do React DOM = virtual DOM

Pasta public é onde o navegador irá procurar os arquivos html, css e js

Todo componente React precisa retornar um elemento jsx

Um componente React é uma função

<> </> = fragmento, só existe no react. Serve para encapsular elementos
